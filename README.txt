Данная программа предназначена для автоматизации работы с сайтом example.ru.
Она позволяет заполнять формы на сайте автоматически и выполнять внеплановые команды
через телеграмм бота. Программа ежедневно отправляет сигнал о работоспособности
и по понедельникам предоставляет отчет за неделю.

Для работы программы необходимо:
1. Постоянное подключение к интернет сети.
2. Операционная система Windows 10.
3. Веб-браузер Google Chrome.

Для запуска программы необходимо разместить представленную для запуска папку в удобной для
Вас директории и запустить через ярлык файл "main.exe". Особенность работы программы состоит
в http соединении. В случае, если необходимо httpS соединение - установите сертификат Selenium Wire CA
(файл "ca.crt" представлен в директории с ярлыком основной программы), добавив его в хранилище
сертификатов "Доверенные корневые центры сертификации" (через путь -> Настройки Chrome ->
Конфиденциальность и безопасность -> Безопасность -> Управление сертификатами устройства -> Импорт... 
ХОТЯ этого и не требуется, если Ваш компьютер не блокирует работу программы. При первом запуске программа 
запросит действия по отладке фонового режима и proxy, остальные действия происходят автоматически.

Для отправки команд через телеграмм бота необходимо ввести команду /help в чате. 
Программа выполнит команды из списка.

Не рекомендуется вносить изменения в установочные файлы и связанные файлы программы.
